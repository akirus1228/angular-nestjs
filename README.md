# nest-angular-starter
This is a repo for a starter appliation for a Single Page MEAN Stack application.

## Installation 
```sh
npm i
```

## Generate client api
```sh
npm run gen-client
```

## Debug with vscode
```sh
npm run debug-server
```
Hit F5 and select the process
